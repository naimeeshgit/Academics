Running the Code

Just run the jupyter notebook file given in the same directory

Contents

The folder contains Reports folder containing the plots showing the phase-space trajectory and the mean squared displacement of the particle.

Results

  • We had initialized the constants like H, k, m from before hand. As seen from the plots, we do get the elliptical path for the phase-space trajectory.
  • For the mean squared displacement plot, since it is only one particle, it becomes a squared displacement plot. It is the shape of a cosing wave.
  • As the value of H gets higher, both the ellipses and cosing waves get larger.
