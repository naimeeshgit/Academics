Results

The prey-predator model has two equations for the growth rate of prey (dN1(t)dt)\bigg(\frac{dN_1(t)}{dt}\bigg)(dtdN1​(t)​) and predator (dN2(t)dt)\bigg(\frac{dN_2(t)}{dt}\bigg)(dtdN2​(t)​). All of these parameters have been fixed.
For the three cases of the prey-predator population, we changed the initial values of N1N_1N1​ and N2N_2N2​ and then solve the Differential equation using Python’s pre-defined libraries.
