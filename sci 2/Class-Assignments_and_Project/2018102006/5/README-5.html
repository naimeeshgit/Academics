Running the Code

Just run the jupyter notebook file given in the same directory

Contents

The folder contains the configuration file for the position of the system of ater molecules at 300 K

Results

  • There are two components: Electrostatic Potential Energy and Van der Waal’s Energy
  • The summation of these two is the total energy for the configuration of the water molecules at 300 K (as given in the pdb file)
  • From the .psf files, we get the charges for Oxygen and Hydrogen atoms. So, I didn’t add that in the folder
