Running the Code

Just run the jupyter notebook file given in the same directory

Contents

The folder contains the plots as asked in the question in the Reports folder.

Results

  • As the temperature increases, the kinetic energy of the particles increases. As a results, we see an increase (negative increase) in the energy in Energy vs Temperature plot
  • Due to the increased kinetic energy, the spin of the particles cannot couple and the magnetization reduces. This is responded by the plot of Magnetization vs Temperature plot
